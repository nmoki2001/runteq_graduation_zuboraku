## サービス概要
- 日付と項目名だけを入力するだけで、自動で家計簿を作成できるシンプル家計簿サービス。
- 過去データを分析し、支出の良い点・改善点を自動で教えてくれる「分析」機能付き（生成AIを利用）。
- 「入力が面倒で家計簿記録ができない人」向けに、必要最低限の入力で家計改善につながることを目指す。
